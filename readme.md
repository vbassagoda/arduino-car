Code for car with 2 servos and 2 motors withan arduino 1 wifi module
We will use a flask app to control the car

Instructions
1) add file secrets.h in the same folder as arduino file:
  #define SSID "[your wifi]"
  #define PASS "[your password]"
2) connect laptop to arduino and upload code
3) run python app.py
4) open html interface in browser in 0.0.0.0:5000
